## **AI System Design**

**Day 1 Fundamentals**
Latency vs throughput, horizontal vs vertical scaling, load balancers, CDN, caching with Redis. Just the concepts, no deep dives.

**Day 2 — Databases + APIs**
When to use SQL vs NoSQL, what vector databases actually do differently, message queues like Kafka for async pipelines.

**Day 3 — RAG design**
Draw the full pipeline on paper. Ingestion → chunking → embedding → indexing → retrieval → reranking → generation. Know why each step exists.

**Day 4 — Agentic system design**
ReAct loop, how tool calling works architecturally, short vs long term memory, what happens when an agent fails mid-task.

**Day 5-6 — Practice on paper**
This is the most important part. Draw while talking out loud. Interviewers care how you think, not just what you know.

**Day 7 — Mock**
Time yourself to 30-40 mins per problem. Defend every decision someone questions.
